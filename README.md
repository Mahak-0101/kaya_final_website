# 🚀 Getting Started

## 1️⃣ Clone the Repository

```bash
git clone https://github.com/your-username/your-repository.git
```

Move into the project folder

```bash
cd your-repository
```

---

## 2️⃣ Install Dependencies

Using npm

```bash
npm install
```

or

```bash
yarn
```

or

```bash
pnpm install
```

or

```bash
bun install
```

---

## 3️⃣ Start Development Server

```bash
npm run dev
```

Open your browser and visit

```
http://localhost:3000
```
