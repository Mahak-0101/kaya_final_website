````markdown
### Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/YOUR_REPOSITORY.git
```

### Move into Project

```bash
cd YOUR_REPOSITORY
```

### Install Dependencies

```bash
npm install
```

```bash
npm run dev
```

Visit

```
http://localhost:3000
````
