# Getting Started

Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/YOUR_REPOSITORY.git
```

Navigate to the project

```bash
cd YOUR_REPOSITORY
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```

Open your browser

```
http://localhost:3000
```

---

# Build for Production

```bash
npm run build
```

Start the production server

```bash
npm start
```
